# горячие клавиши
#### выделить все:
Ctrl+A
#### копирывать:
Ctrl+C
#### вставить:
Ctrl+V
#### вырезать:
Ctrl+X
#### отменить:
Ctrl+Z
#### созранить:
Ctrl+S

# работа с git
### скачать проект:
```bash
git clone <URL>
```
### учесть все файлы:
```bash
git add .
```
### сохранить комит:
```bash
git commit -m "..."
```
### отправка в облоко:
```bash
git push
```
### принять изменинея из облака :
```bash
git pull
```
